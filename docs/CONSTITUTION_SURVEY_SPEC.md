# ここから。体質アンケート仕様書

## 目的

体質アンケートは、医療診断ではなく**今の体と生活の反応傾向**を見るためのものとして扱う。

目指すことは次の4つ。

1. 少しでも「当たってる」と感じられること
2. 何度かやっても違いが見えること
3. 結果がその後の伴走に活きること
4. 合いやすいAIタイプ提案へ自然につながること

---

## UI方針

- LINEで押すだけで答えられる
- 1問1画面
- 横スライドなし
- 選択肢は1画面で全部見える
- 初回アンケートも定期チェックも**3択**

### 初回アンケートの回答
- 少ない
- ときどきある
- よくある

### 定期チェックの回答
- 整っている
- いつも通り
- 崩れ気味

---

## 全体構造

### 1. 初回アンケート
- 12問
- 主タイプ + 副タイプを返す
- その人のベース傾向を見る

### 2. 定期チェック
- 7問
- 前回との差分コメントを返す
- 今の整い具合や揺れを見る

---

## 結果タイプ 8種

### 1. fluid = むくみためこみ型
重さやむくみ、冷え感として出やすい傾向。

### 2. slow = 省エネ停滞型
朝の重さや立ち上がりの鈍さとして出やすい傾向。

### 3. overwork = 消耗がんばり型
頑張れてしまうぶん、気づくと消耗が大きい傾向。

### 4. hold = 我慢ためこみ型
つらさや違和感を言葉にせず抱え込みやすい傾向。

### 5. sweet = 甘いもの波型
甘いものや炭水化物に引っ張られやすい食欲の波が出る傾向。

### 6. foggy = 食後どんより型
食後の眠さ、重さ、ぼんやり感が出やすい傾向。

### 7. stress = 気疲れゆらぎ型
人や空気に気をつかい、その負荷が体調や生活に出やすい傾向。

### 8. overthink = 考えすぎ停止型
考えすぎて整理できず、動き出しにくくなる傾向。

## 初回アンケート設問

1. 冷えやむくみを感じやすいですか？
2. 朝、体が重いと感じやすいですか？
3. 疲れが抜けにくいと感じますか？
4. 無理すると数日ひびきやすいですか？
5. 食後に眠い・重いと感じやすいですか？
6. 甘いものがやめにくいと感じますか？
7. ストレスで食欲が乱れやすいですか？
8. つらくても我慢しやすいですか？
9. 人に気をつかいすぎて疲れますか？
10. 考えすぎて動けなくなることはありますか？
11. 首・肩・腰が固まりやすいですか？
12. 忙しいと生活が乱れやすいですか？

---

## 採点の考え方

### 回答点
- 少ない = 主0 / 副0
- ときどきある = 主1 / 副0
- よくある = 主2 / 副1

### 設問ごとの主タイプ / 副タイプ

| 設問 | 主タイプ | 副タイプ |
|---|---|---|
| Q1 | fluid | slow |
| Q2 | slow | fluid |
| Q3 | overwork | slow |
| Q4 | overwork | hold |
| Q5 | foggy | slow |
| Q6 | sweet | foggy |
| Q7 | sweet | stress |
| Q8 | hold | overthink |
| Q9 | stress | hold |
| Q10 | overthink | stress |
| Q11 | hold | fluid |
| Q12 | stress | overwork |

### 結果の出し方
- 合計点が最も高いものを主タイプ
- 次点を副タイプ
- 同点時は固定のキー順で処理してもよいが、将来的には生活傾向や直近入力で補正してもよい

## 結果文の構成

各タイプは次の要素を持つ。

- summary
- body
- signs
- relief
- tips
- closing
- recommended_personas

### 返し方の原則
- 診断のように断定しない
- 今の傾向として伝える
- 不安をあおりすぎない
- 最後は変化可能性で閉じる

### 基本フォーマット
1. 主タイプ + 副タイプの見出し
2. 主タイプ summary / body
3. relief
4. 副タイプの短い補足
5. 前向きな closing

## 定期チェック設問

1. 最近、朝の体の軽さはどうですか？
2. 最近、疲れの抜けやすさはどうですか？
3. 最近、食後の楽さはどうですか？
4. 最近、食欲の安定感はどうですか？
5. 最近、体のこわばりはどうですか？
6. 最近、気持ちや生活リズムはどうですか？
7. 最近、頑張りすぎている感じはどうですか？

---

## 定期チェックの見方

### 回答点
- 整っている = -1
- いつも通り = 0
- 崩れ気味 = +1

### 用途
- 初回アンケートの固定化を避ける
- 直近の波を見えるようにする
- 前回との変化コメントにつなげる

### 差分コメント例
- 前回より、気持ちや生活リズムの揺れは少し落ち着いています。
- 今週は、頑張りすぎの傾向が少し強めです。減速を優先してもよさそうです。
- 食後の重さやぼんやり感が少し出やすいかもしれません。

---

## AIタイプとの接続

結果は、その後の伴走に自然につなげる。

### 例
- stress / hold が強い → そっと寄り添う
- overthink / slow が強い → 頼もしく導く
- sweet / foggy が強い → 明るく後押し
- overwork / hold が強い → 力強く支える または そっと寄り添う

ただし、決めつけではなく**合いやすい提案**として扱う。
